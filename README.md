Final Project

#####################
Running the program
#####################
connect4.py --a [algorithm] --p [player]

Argument description:
algorithm - [MonteCarlo, NN, QL], this is your opponent
player - [1, 2], choose player 1 or 2

Example:
python3 connect4.py --a NN --p 1
